Zabbix5.0实现钉钉告警

一、准备钉钉环境

1.1 安装pc版钉钉

钉钉下载地址：https://www.dingtalk.com/

下载，安装，注册，登陆，过程略

1.2 创建群聊

点击右上方的“+”号按钮

点击“发起群聊”

选择“内部项目群”（创建任意群都行）

填写配置，并点击创建

1.3 创建钉钉机器人

点击“群设置”

点击“群智能助手”

点击“添加机器人”

点击“+”号

选择“自定义”

点击“添加”

添加机器人->点击“完成”

获取机器人的webhook，点击“设置说明”可以查看钉钉服务端API说明。

二、配置告警脚本

2.1 查看zabbix-server配置文件中告警脚本存放位置

找到报警脚本存放路径，取消注释，然后修改路径（建议写成绝对路径）

修改完成后重启zabbix-server

2.2 添加存放脚本的文件夹

2.3 添加告警脚本

三、添加报警媒介类型

在web主页点击“管理”->“报警媒介类型”

点击“创建媒体类型”

添加配置，然后点击添加

三个参数：{ALERT.SENDTO} {ALERT.SUBJECT} {ALERT.MESSAGE}

在新添加的媒体类型点击“测试”

收件人填写手机号码，然后点击测试

如果文件存在会显示“Media type test successful.”，如果位置刚才的脚本放的位置不对 这里会提示没有此脚本的

四、创建用户组

点击“管理”->“用户群组”->“创建用户群组”

根据配置填写，然后点击添加

点击“权限”

注意：如果用户组没有主机群组的权限将不能发送告警消息！！！

选择需要添加的主机群组（主机群主指的是在添加主机时，需要选择的一个群组），然后点击添加

添加主机群组后点击更新（或添加）

五、创建用户

点击”管理”->“用户”->“创建用户”

根据配置填写

点击“报警媒介”->点击“添加”

按着配置填写，然后点击添加

六、添加动作

点击“配置”->“动作”

点击“创建动作”

填写配置信息，然后点击“操作”

点击“操作”->在操作里点击“添加”

根据配置填写

Custom message内容

恢复操作点击“添加”

根据需求填写，然后点击添加

在添加了操作和恢复操作后点击添加

七、测试

将服务器的MySQL关闭，等待一下

再将服务器的MySQL启动，等待一下
